# Cupertino Room Rental Listing Project

## Summary
Created and deployed a professional rental listing webpage for a furnished room in Cupertino, CA.

**Live URL:** https://floralu2015.github.io/cupertino-rental/rental_listing.html

---

## What We Did

### 1. Image Processing
- Read and reviewed photos from `/basement room pictures/` folder
- Compressed all images from **64MB to 8MB** for faster loading
- Organized images into categories: Bedroom, Bathroom, Office, Kitchen, Exterior, etc.
- Created optimized folder: `/basement_room_optimized/`

### 2. HTML Listing Creation
- Built a professional rental listing HTML page
- Added photo gallery with lightbox functionality
- Organized sections: Property Highlights, Photo Gallery, Room Details, House Rules, Contact

### 3. Design Enhancements
- Added full-screen **hero banner** with house exterior image
- Implemented **Google Fonts** (Playfair Display + Inter)
- Added smooth **scroll animations** as sections come into view
- Created **hover effects** on cards and images
- Added **floating "Contact Now" button**
- Applied **soft gradient background** (teal → cyan → lavender)

### 4. Hosting Setup
Tried multiple hosting options:

| Service | Result |
|---------|--------|
| Netlify | Account site limit exceeded |
| Surge | Required login |
| Vercel | Required login |
| **GitHub Pages** | Successfully deployed |

**Final hosting:** GitHub Pages via repository `floralu2015/cupertino-rental`

### 5. Fair Housing Compliance
- Removed "Ideal Tenant" section (potential discrimination concerns)
- Replaced with "House Rules" section focusing on property rules:
  - Smoke-Free Property
  - No Pets Allowed
  - Quiet Hours 10PM-8AM
  - Shared Spaces Kept Tidy
  - No Overnight Guests Policy
  - Background Check Required

### 6. Marketing Copy Created
Created listing descriptions for:
- Craigslist (long version)
- Facebook Marketplace (short version)
- LinkedIn (professional network post)
